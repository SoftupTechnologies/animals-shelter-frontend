# Protect me Albania - Animals Shelter Database

## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Run Production Build Locally

Run build

```bash
  npm run build
```

Navigate to the build folder

```bash
  cd build
```

Start the http-server

```bash
  http-server --push-state
```
